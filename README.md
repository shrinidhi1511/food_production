# Digital Twin Food Quality System

A modern React + Vite dashboard designed for monitoring and optimizing food production operations using real-time analytics, AI-powered insights, and digital twin concepts.
This project simulates a digital twin system for food manufacturing environments. It helps monitor production quality, sensor activity, batch performance, and AI-assisted predictions through a centralized dashboard interface.

## Live Demo

https://foodproduction.vercel.app

---

## Features

- Real-time production monitoring dashboard
- AI-powered quality prediction cards
- Sensor analytics and batch tracking
- Recipe optimization section
- Responsive sidebar navigation
- KPI metrics visualization
- Interactive charts and data panels
- Clean modern UI with Tailwind CSS

---

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript
- Context API

## Installation

Clone the repository:

```bash
git clone https://github.com/shrinidhi1511/food_production.git
```

Navigate into the project:

```bash
cd food_production
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
